# PathExplorer Backend

This is the backend API for the PathExplorer system, developed for Accenture's Monterrey office.

## Technologies Used

- Node.js
- TypeScript
- Express.js
- Supabase (PostgreSQL)
- JWT Authentication

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Supabase account
